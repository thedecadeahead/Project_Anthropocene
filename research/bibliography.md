# Master Bibliography

This file indexes the major sources used across Project Anthropocene.

Detailed notes should live in domain-specific research files. The master bibliography exists to prevent duplicate work, standardize citations, and show which sources materially influence canon.

## Citation Format

Use the following structure when practical:

```text
Author or Institution. "Title." Publication or Report Series, year. DOI, report number, or stable URL. Accessed YYYY-MM-DD.
```

For datasets, include the version or release date.

For reports, include the chapter, page, table, or figure used when relevant.

## Source Record Template

```markdown
### [Short Source Name]

- **Full citation:**

- **Used in:**
- **Limitations:**
- **Last checked:** YYYY-MM-DD
```

## Status Tags

Use these tags after a source heading when useful:

- `[FOUNDATIONAL]` — central synthesis or dataset used across the project.
- `[DOMAIN]` — important within a specific research area.
- `[LEAD]` — promising source not yet fully reviewed.
- `[SUPERSEDED]` — replaced by a newer edition or stronger source.
- `[DISPUTED]` — findings are materially contested or difficult to reproduce.

## Bibliography Maintenance

- Prefer stable identifiers over temporary links.
- Note when a source has been superseded, but do not silently remove it if canon previously relied upon it.
- Record the date a web source was last checked.
- Link every source to the research notes that use it.
- Avoid placing a source in multiple sections; cross-reference it instead.
- Do not treat inclusion in this bibliography as endorsement of every claim in a source.
