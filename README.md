# Project Anthropocene

> *The future has not been written yet.*

Project Anthropocene is an evidence-informed collaborative worldbuilding project exploring one plausible future of Earth following an inadequate global response to climate change.

This is not a prediction. It is a rigorously researched future history intended to support novels, tabletop roleplaying games, video games, and collaborative storytelling.

## Core Principles

- Evidence before speculation.
- Every major event should have a plausible chain of causes.
- Technology continues to advance even as institutions struggle.
- Humanity adapts.
- Hope survives.

Contributors are referred to as **Researchers**. The goal is to construct a believable future history, not simply write fiction.
